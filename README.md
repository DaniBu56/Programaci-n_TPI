*Descripción del Programa*
Este proyecto es una aplicación en Python que gestiona información de países, permitiendo al usuario consultar y editar un dataset previamente creado o agregar datos nuevos. Utiliza listas, donde la principal se usa para guardar países y diccionarios para almacenar datos de los países con  4 claves: "nombre", "poblacion", "superficie", "continente". Lee un archivo CSV y permite realizar operaciones desde un menú en consola.
Es un código modularizado con una función por opción de ejecución del bloque principal "menú" y algunas funciones auxiliares, principalmente para ordenar y estadísticas.

*Instrucciones de Uso*
1. Asegurarse de tener el archivo "Datos_paises.csv" en la misma carpeta que el programa.
2. Ejecutar el archivo principal python "TPI entregable.py".
3. Usar el menú para elegir las opciones disponibles:
   /1. Agregar país
    2. Actualizar país
    3. Buscar país
    4. Filtrar países
    5. Ordenar países
    6. Mostrar estadísticas
    7. Salir/
